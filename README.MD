## üå± Developer Hygiene Automation

This repository includes a **self-maintaining developer hygiene system** powered by GitHub Actions and Dependabot.  
It ensures consistent formatting, testing, and configuration management automatically.

| üß© Workflow | üß∞ Purpose | ‚è±Ô∏è Schedule |
|:------------|:-----------|:------------|
| üßº **Format (`format.yml`)** | Runs Black + pre-commit formatters on all PRs. Auto-commits fixes or leaves comments on forks. | Every PR |
| üßπ **Bootstrap Dev Hygiene (`bootstrap-dev-hygiene.yml`)** | Ensures configuration files (`pyproject.toml`, `.pre-commit-config.yaml`, `.editorconfig`, `requirements-dev.txt`, etc.) are present and correct. | Weekly (Monday 06:00 UTC) |
| üîÅ **Auto-Update Pre-commit (`auto-update-precommit.yml`)** | Refreshes all hook versions and opens a PR with updates. | Monthly (1st of each month) |
| ü§ñ **Dependabot (`dependabot.yml`)** | Keeps `requirements-dev.txt` and GitHub Actions versions up-to-date. | Weekly |
| üß© **Branch Protection + CODEOWNERS** | Ensures all PRs pass checks and assigns appropriate reviewers automatically. | Continuous |
